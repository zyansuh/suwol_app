# 카페링크 수월 (CafeLink Suwol)

**슬로건**: 우리 동네 카페들이 하나로 이어지다

## 프로젝트 소개

독립 카페들이 함께 고객 혜택을 공유하고, 소비자는 연합 멤버십을 통해 다양한 카페를 즐길 수 있는 상생 플랫폼입니다.

## 서비스 대상자

- 자영업 카페 운영자
- 지역 카페 이용 소비자

## 핵심 가치

- 독립 카페들이 함께 고객 혜택을 공유
- 소비자는 연합 멤버십을 통해 다양한 카페를 즐길 수 있음
- 하나의 앱으로 모든 제휴 카페의 포인트, 쿠폰을 통합 관리
- 소비자는 "통합 포인트"를 적립/사용
- 카페 사장님은 "연합 마케팅" 및 신규 고객 유입 효과

## 주요 기능

1. **통합포인트/쿠폰 시스템**
2. **카페 맵 & 탐색 기능**
3. **고객 리워드 & 등급제**
4. **사장님 전용 관리 페이지**
5. **커뮤니티 피드**

## 프로젝트 구조

```
lib/
├── main.dart
├── app.dart
├── models/          # 데이터 모델
├── screens/         # 화면
├── widgets/         # 재사용 가능한 위젯
├── services/        # API, 로컬 스토리지 등
├── providers/       # 상태 관리
├── utils/           # 유틸리티 함수
├── constants/       # 상수
└── theme/           # 테마 설정
```

## 시작하기

```bash
flutter pub get
flutter run
```
